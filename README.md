SmartBudget 💰

Prosta aplikacja webowa do zarządzania budżetem domowym, napisana w Pythonie (framework Flask).
Pozwala na dodawanie przychodów i wydatków, przeglądanie historii transakcji oraz automatycznie oblicza aktualne saldo konta.

Jak uruchomić projekt lokalnie?
1. Upewnij się, że masz zainstalowanego Pythona.
2. Zainstaluj wymagane biblioteki wpisując w terminalu: `pip install -r requirements.txt`
3. Uruchom główny plik aplikacji: `python app.py`
4. Otwórz w przeglądarce podany link (zazwyczaj `http://127.0.0.1:5000`).

Wykorzystanie AI
W ramach realizacji projektu AI pomogło nam w wygenerowaniu struktury testów w pytest oraz wsparło nas w responsywnym formatowaniu tabeli HTML.
